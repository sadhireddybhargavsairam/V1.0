IOT based Smart weather monitoring system using STM32 micro controller.

In here we are gonna read Tempeature from sensor and send the value to kernel masters server for ever 5 sec and simultaneously print the value in LCD


Hardware used : Raayan mini https://github.com/raayanmini/Raayanmini-Docs.git
Server : http://142.93.218.33/km/weathermonitor.php
